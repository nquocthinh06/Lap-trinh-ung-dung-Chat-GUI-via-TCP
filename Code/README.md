# Mã nguồn

Thư mục này chứa toàn bộ mã nguồn của ứng dụng Chat (TCP).

* **Client/**, **Server/**, **Shared/** — mã C++.
* **CMakeLists.txt**, **CMakePresets.json** — build bằng CMake (Windows).
* **.vscode/** — gợi ý extension & debug cho VS Code *(mở **folder `Code`** trong IDE)*.

Hướng dẫn cài đặt, build và chạy: xem **README.md** ở thư mục gốc repo.
